Mac常用软件推荐

Author: Jason Date: November 29, 2018

介绍一些Mac平台常用的工具。

效率工具

  1. Evernote
  2. Microsoft Todo
  3. Vitamin-R
  4. Trello

阅读工具

  1. Reeder/NewsBlur：RSS
  2. Pocket：Read it later
  3. iBook

写作工具

  1. Typora：最好用的mark down编辑器，没有之一
  2. Alternote：连接EverNote的笔记应用
  3. iThoughtsX / Xmind ：思维导图
  4. OmniGraffle

开发工具

  1. SourceTree/Cornerstone：版本管理工具
  2. RubyMine/PyCharm/AppCode/VS Code/Xcode/CodeFlow/Sublime Text
  3. Docker
  4. Base：数据库
  5. Knuff：推送测试
  6. Understand：静态分析
  7. Reveal
  8. ProtobufViewer：ProtoBuf查阅
  9. MachOView
  10. Hopper Disassembler/iDA Pro：反汇编
  11. Cyberduck：加密FTP
  12. Charles/Wireshark：代理&抓包
  13. Postman：模拟请求

通讯工具

  1. Shadowsocks
  2. Telegram
  3. Gitter

设计工具

  1. Sketch
  2. PxCook 设计师软件
  3. Eagle
  4. 蓝湖

日常工具

  1. Paste：剪贴板
  2. Sip：颜色提取
  3. Kap：简单录屏
  4. Xnip：截图
  5. Skim：PDF阅读
  6. The Unarchiver：压缩工具
  7. Elmedia Player：播放器
  8. Alfred 3：快捷工具
  9. GPG
  10. resilio sync
